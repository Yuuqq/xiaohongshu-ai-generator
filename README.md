# 🎨 小红书AI图片生成器

一个基于Google Gemini API的智能图片生成工具，专为小红书内容创作者设计。

## ✨ 功能特性

- 🤖 **AI驱动**: 集成Google Gemini API，智能生成高质量图片
- 🎯 **模板丰富**: 多种小红书风格模板，满足不同场景需求
- 🎨 **高级设计**: Google/Claude工程师级别的UI设计
- 📱 **响应式**: 完美适配桌面端和移动端
- 📦 **批量下载**: 一键下载所有生成的图片
- ⚡ **实时预览**: 即时查看生成效果

## 🚀 快速开始

1. 打开 `index.html` 文件
2. 输入您的Google Gemini API密钥
3. 输入要生成图片的文字内容
4. 选择喜欢的模板风格
5. 点击生成，等待AI创作
6. 预览并下载生成的图片

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **API**: Google Gemini API
- **设计**: Material Design 3.0
- **字体**: Google Fonts
- **图标**: Material Icons
